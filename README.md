# Aldeia Indígena Frontend

Projeto frontend desenvolvido com React + Vite para o sistema Aldeia Indígena.

## Status
Em desenvolvimento

## Tecnologias
- React
- Vite
- JavaScript
- CSS (Global)

## Estrutura do Projeto
- src/components → Componentes reutilizáveis
- src/pages → Páginas da aplicação
- src/styles → Estilos globais

## Como rodar o projeto

```bash
npm install
npm run dev
